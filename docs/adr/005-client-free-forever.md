# ADR-005: The client is free forever

**Status:** Accepted

## Decision

The extension contains no license gates, no feature clawbacks, no telemetry, and no monetization — ever (invariant P8). If a managed offering ever exists, it is a separate product that this client does not depend on.

## Rationale

The user already pays for inference through their own OpenRouter key at cost. Charging again in the client, or holding features hostage, would recreate exactly the SaaS dynamics the project exists to invert. Stating it as an ADR + README commitment makes reversal a visible governance event, not a quiet release note.
